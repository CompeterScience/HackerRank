# Basics of Sets and Relations

### Question #1
You are given two sets.  
`Set A = {1,2,3,4,5,6}  
Set B = {2,3,4,5,6,7,8}`
How many elements are present in A∪B?  
### Answer
8

### Question #4
You are given two sets.  
`Set A = {1,2,3,4,5,6}
Set B = {2,3,4,5,6,7,8}  `

### Answer
42
